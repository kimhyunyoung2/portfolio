# 스크롤에 반응하는 헤더 스타일 변경 구현 사용자가 페이지를 아래로 스크롤할 때 헤더의 배경색과 크기가 동적으로 조절되도록 구현한 예제입니다. ## 구현 세부사항 ### HTML 구조 - 상단에 고정된 헤더(`#main-header`)를 포함하고 있으며, 내부에 메뉴 항목을 구성하는 `nav ul li` 요소가 있습니다. ### JavaScript 동작 - 스크롤 이벤트를 감지하여 사용자가 일정 높이 이상 스크롤하면 헤더에 `.scrolled` 클래스를 추가해 스타일을 변경합니다.
