# 드롭다운 메뉴와 서브메뉴 구현 드롭다운 메뉴와 서브메뉴 구현 예제에서는 사용자 인터페이스에 통합할 수 있는 방법을 설명합니다. ## 구현 세부사항 ### HTML 구조 - 드롭다운 메뉴(`.menu`), 메뉴 아이콘(`.menu-icon`), 메뉴 본체(`.dropdown-content`), 서브메뉴(`.submenu`)로 구성됩니다. ### JavaScript 동작 - `.menu-icon` 클릭 시 메인 메뉴의 표시 상태를 토글하고, `.menu-item` 클릭 시 연결된 서브메뉴의 표시 상태가 토글됩니다.
