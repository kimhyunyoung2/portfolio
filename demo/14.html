# To Do List 관리 구현 사용자가 할 일을 추가하고, 할 일의 완료 여부를 표시하며, 필요 없는 할 일을 삭제할 수 있는 간단한 웹 애플리케이션을 구현했습니다. 모든 To Do List 브라우저의 로컬 스토리지에 저장되어, 사용자가 애플리케이션을 다시 방문할 때 이전 상태를 복원할 수 있습니다. ## 구현 세부사항 ### HTML 구조 - 할 일을 입력할 수 있는 입력 필드와 추가 버튼이 포함된 입력 그룹, 추가된 To Do List 표시하는 리스트가 있으며, 각 항목 옆에는 완료 및 삭제 버튼이 있습니다. ### JavaScript 동작 - 할 일을 추가하는 기능, To Do List를 로컬 스토리지에 저장하는 기능, 페이지 로드 시 저장된 To Do List 불러오는 기능이 구현되어 있습니다. 각 To Do List 옆의 완료 버튼을 클릭하면 해당 할 일이 완료 상태로 토글되고, 삭제 버튼을 클릭하면 해당 항목이 목록에서 제거됩니다. 입력 필드에서 Enter 키를 눌러도 할 일이 추가됩니다.
