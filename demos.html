모달 구현

HTML, CSS, 그리고 JavaScript를 활용해 사용자에게 정보를 제공하거나 추가 입력을 받을 수 있는 모달(dialog)을 구현하는 방법을 보여줍니다.

슬라이드 기능

슬라이더 기능을 통해 사용자가 이미지나 내용을 좌우로 슬라이딩할 수 있습니다.

반응형 탭 인터페이스

서로 다른 정보 섹션을 탐색할 수 있는 반응형 탭 인터페이스를 구현합니다.

드롭다운 메뉴

드롭다운 메뉴와 서브메뉴를 구현하여 사용자 인터페이스를 더욱 편리하게 사용할 수 있도록 작업했습니다.

계산기

웹 페이지에 간단한 계산기를 구현해서 기본적인 사칙연산을 수행할 수 있습니다.

무한 스크롤

웹 페이지에 무한 스크롤 기능을 추가하여, 사용자가 스크롤할 때마다 자동으로 콘텐츠가 로드되도록 구현했습니다.

고정 헤더

사용자가 페이지를 아래로 스크롤할 때 헤더의 배경색과 크기가 동적으로 조절되도록 구현했습니다.

비밀번호 강도 검증 기능

사용자가 입력한 비밀번호의 안전성을 실시간으로 검증하고, 그 결과를 피드백으로 제공하는 기능을 구현했습니다. 사용자는 비밀번호 강도에 따라 적절한 메시지와 색상으로 된 설명을 받게 됩니다.

이미지 지연 로딩 기능

웹 페이지 내 이미지 리소스의 로딩을 지연시키는 기능을 구현하여 초기 페이지 로드 시간을 개선했습니다

쇼핑 카트 기능

사용자가 상품을 쇼핑 카트에 추가하고 카트 내의 상품을 관리할 수 있는 기능을 구현했습니다.

달력 기능

Flatpickr 라이브러리를 활용하여 사용자가 달력 UI를 통해 시작일과 종료일을 선택할 수 있는 날짜 선택 기능을 구현했습니다.

인터랙티브 카드 효과

사용자의 마우스 움직임에 따라 동적으로 스타일이 변화하는 인터랙티브한 카드 디자인을 구현했습니다.

이미지 자르기 기능

Cropper.js 라이브러리를 활용하여 사용자가 이미지를 선택하고 크롭할 수 있는 기능을 구현했습니다. 크롭된 이미지의 데이터 URL은 사용자가 복사할 수 있습니다.

할 일 목록 기능

사용자가 할 일을 추가하고, 할 일의 완료 여부를 표시하며, 필요 없는 할 일을 삭제할 수 있는 간단한 웹 애플리케이션을 구현했습니다. 모든 To Do List 브라우저의 로컬 스토리지에 저장되어, 사용자가 웹 페이지를 다시 방문할 때 이전 상태를 확인할 수 있습니다.

패럴랙스 스크롤링 효과

패럴랙스 스크롤링 효과를 구현했습니다. 사용자가 스크롤할 때 배경 이미지가 고정되어 있으며, 내용 텍스트만 움직이는 효과를 보여줍니다.

일주일 동안 보지 않기

로컬 스토리지를 이용해 "일주일 동안 보지 않기" 기능을 구현하여 사용자 경험을 개선했습니다.

카테고리 통한 상품 추천

사용자가 선호하는 카테고리를 선택할 수 있게 하고, 선택에 따라 해당 카테고리에 맞는 상품 이미지를 추천해 주는 기능을 구현했습니다.

상품 필터링 및 정렬 기능

사용자가 선호하는 카테고리별로 상품을 필터링하고, 가격 순으로 정렬할 수 있는 웹 페이지 기능을 구현했습니다.

고객 리뷰 및 평점

고객 리뷰와 평점 기능을 구현했습니다.
