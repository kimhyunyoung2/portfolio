# 패럴랙스 스크롤링 구현 패럴랙스 스크롤링 효과를 구현했습니다. 사용자가 스크롤할 때 배경 이미지가 고정되어 있으며, 내용 텍스트만 움직이는 효과를 보여줍니다. ## HTML 구조 HTML 문서는 기본적으로 두 부분으로 구성됩니다: `.parallax`와 `.content`. - `.parallax`: 패럴랙스 배경을 위한 `div`입니다. `background-image`를 통해 배경 이미지를 설정하고, `background-attachment: fixed;` 속성을 사용하여 스크롤 시에 배경이 고정되도록 합니다. - `.content`: 실제 내용을 담고 있는 `div`입니다. 이 부분에는 제목과 설명이 포함되어 있으며, 사용자가 스크롤할 수 있도록 충분한 높이를 가지고 있습니다. ## CSS 스타일 - `.parallax`에 적용된 `background-attachment: fixed;` 속성을 주었고, 배경 이미지는 스크롤과 독립적으로 고정됩니다. - `.content`는 사용자가 실제로 인터랙션할 내용을 담고 있으며, `height: 2000px;`로 설정하여 스크롤이 발생하도록 합니다. 배경색은 `rgba(255, 255, 255, 0.8);`로 설정하여 배경 이미지가 비치도록 합니다.

배경과 내용이 다른 속도로 움직입니다. 스크롤을 내려서 효과를 확인해보세요.
