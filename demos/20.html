# Tailwind CSS 상품 리스트 Tailwind CSS을 사용해 상품 리스트를 구현했습니다. ### 구현 포인트 - **스타일링:** Tailwind CSS CDN을 통해 적용했습니다. - **레이아웃:** `.container`와 Tailwind의 그리드 시스템을 사용해 반응형 상품 카드 레이아웃을 구성했습니다. - **디자인:** `rounded-lg`, `shadow-lg`, `overflow-hidden` 등의 클래스로 상품 카드에 둥근 모서리, 그림자 효과를 추가하고 이미지는 `object-cover`로 처리해 꽉 찬 비율을 유지되도록 구성했습니다. - **상품 정보:** 시맨틱 태그와 `.ttruncate`를 사용해 상품 이름, 가격, 설명을 표시하며, 설명은 3줄로 제한했습니다. - **인터랙션:** `bg-yellow-500`, `hover:bg-yellow-600` 클래스로 "장바구니에 추가" 버튼에 인터랙티브한 효과를 적용했습니다.
