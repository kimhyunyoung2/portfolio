# 카테고리 선택을 통한 상품 추천 UI 사용자가 선호하는 카테고리를 선택할 수 있게 하고, 선택에 따라 해당 카테고리에 맞는 상품 이미지를 추천해 주는 기능을 구현했습니다. ## 구현 세부사항 JavaScript를 통해 "카테고리 선택을 통한 상품 추천" 기능을 구현했습니다. ### JavaScript - **카테고리 선택:** 사용자가 드롭다운 메뉴(`categorySelect`)에서 카테고리를 선택합니다. - **추천 보기 버튼 활성화:** 사용자가 '추천 보기' 버튼을 클릭하면, `showRecommendations` 함수가 호출됩니다. - **동적 상품 추천 표시:** - 함수는 먼저 `recommendations` 영역의 내용을 초기화합니다. - 선택된 카테고리에 해당하는 상품 이미지 경로가 담긴 배열에서 상품을 불러옵니다. - 각 상품에 대해 HTML `div` 요소를 생성하고, `img` 요소에 상품 이미지를 할당한 후, 이를 `recommendations` 영역에 추가합니다.
