# Flatpickr를 이용한 날짜 선택기 구현 Flatpickr 라이브러리를 활용하여 사용자가 달력 UI를 통해 시작일과 종료일을 선택할 수 있는 날짜 선택 기능을 구현했습니다. ## 구현 세부사항 ### HTML 구조 - 시작일과 종료일을 위한 두 개의 `input` 필드가 제공되며, 사용자는 이 입력란을 통해 날짜를 선택할 수 있습니다. '확인' 버튼을 클릭하면 선택된 날짜가 다른 입력란에 표시됩니다. ### JavaScript 동작 - `flatpickr` 함수를 사용하여 날짜 선택기 기능을 초기화하고, 시작일 선택 시 종료일 선택기의 최소 선택 가능 날짜를 설정합니다. 반대로 종료일 선택 시 시작일 선택기의 최대 선택 가능 날짜를 설정합니다. '확인' 버튼 클릭 시 선택된 날짜가 표시됩니다.
