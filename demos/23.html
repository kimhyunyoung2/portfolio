# 스크롤 프로그레스 바 스크롤하면 페이지 상단에 위치한 프로그레스 바가 채워지는 예제를 구현했습니다. ### 구현 포인트 - 페이지에 스크롤 이벤트 리스너를 추가하여, 사용자가 페이지를 스크롤할 때마다 updateProgressBar 함수를 호출합니다. - updateProgressBar 함수는 현재 스크롤 위치를 계산하고, 전체 페이지 높이에 대한 스크롤된 비율을 기반으로 프로그레스 바의 너비를 조정합니다. - 프로그레스 바는 사용자가 페이지에서 얼마나 스크롤 했는지를 시각적으로 표시해 줍니다.

스크롤하면 프로그레스 바가 채워집니다
