# 이미지 지연 로딩(Lazy Loading) 기능 구현 웹 페이지 내 이미지 리소스의 로딩을 지연시키는 기능을 구현하여 초기 페이지 로드 시간을 개선했습니다. ## 구현 세부사항 ### HTML 구조 - 지연 로딩을 적용할 이미지들을 `img.lazyload` 클래스와 함께 `data-src` 속성에 실제 이미지 URL을 저장하여 배치했습니다. 이를 통해 초기 로딩 시에는 이미지 대신 가벼운 이미지가 로드됩니다. ### JavaScript 동작 - Lazy Load 라이브러리를 사용하여 `DOMContentLoaded` 이벤트가 발생하면 지연 로딩을 적용했습니다. 사용자가 스크롤하여 이미지가 보이는 부분에 도달하면, 실제 이미지가 로드되어 사용자에게 표시됩니다.
